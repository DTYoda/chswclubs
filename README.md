# 📚 CHSW Clubs – CS50x Final Project  

![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)
![Flask](https://img.shields.io/badge/Flask-000000?style=for-the-badge&logo=flask&logoColor=white)

**CHSW Clubs** is a club social media platform created in **Flask/Python** as my final project for **Harvard’s CS50x**.  
Its goal is to provide **Cranston High School West** students with a single website where they can:  
- Discover and join clubs  
- Connect and socialize with other members  
- Stay updated with club notifications  

The project is currently in the **prototype phase**, with core functionality working and visuals under active development.  

---

## ✨ Features  
✅ User authentication (login & registration)  
✅ Create a new club  
✅ Join existing clubs  
✅ Search clubs by name  
✅ Club messaging & updates feed  
🚧 UI/UX visuals are still in development  

---

## 🛠️ Tech Stack  
- **Backend:** Flask (Python)  
- **Database:** SQLite  
- **Frontend:** Flask’s Jinja2 templating (HTML injection)  
- **Other Tools:** Git, CS50 Library
